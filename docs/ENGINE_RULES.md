# Engine Rules

## Regra 001

O Core nunca altera dados.

Ele apenas calcula.

---

## Regra 002

Toda informação deve possuir uma única origem.

(Single Source of Truth)

---

## Regra 003

Interfaces nunca executam cálculos.

---

## Regra 004

Toda fórmula deve possuir pelo menos um teste.

---

## Regra 005

Nenhuma recomendação pode existir sem documentação.

---

## Regra 006

O projeto deverá permanecer independente da interface.

Excel, Python, Web ou Mobile utilizarão exatamente o mesmo Engine.

---

## Regra 007

Nenhum número entra no sistema sem justificativa técnica.
