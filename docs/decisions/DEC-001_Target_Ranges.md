# DEC-001 — Faixas Nutricionais do MVP

## Decisão

O Calc-Grower Pro utilizará faixas de consenso para cultivo em fibra de coco como referência inicial do módulo Target Engine.

## Motivo

As faixas representam a convergência entre literatura científica, manuais técnicos de hidroponia e prática consolidada em coco.

## Valores centrais adotados

* N = 175 ppm
* P = 55 ppm
* K = 225 ppm
* Ca = 175 ppm
* Mg = 60 ppm
* S = 80 ppm

## Status

Aceita.

## Revisão

Poderá ser revisada após validação experimental com dados reais de cultivo.
