# Calc-Grower Pro

Versão da Especificação

v1.0 Draft

---

# Objetivo

O Calc-Grower Pro é um sistema de engenharia nutricional destinado ao desenvolvimento, validação e gerenciamento de soluções nutritivas para cultivos em substratos inertes.

O projeto prioriza:

- precisão matemática;
- rastreabilidade dos cálculos;
- arquitetura modular;
- documentação completa;
- evolução contínua.

---

# Escopo do MVP

A primeira versão deverá permitir:

1. Cadastro de fertilizantes.
2. Cadastro da água.
3. Criação de receitas.
4. Cálculo automático de macro e micronutrientes.
5. Estimativa de EC.
6. Comparação com metas.
7. Geração de relatório.

---

# Fora do Escopo

Não fazem parte do MVP:

- balanceamento automático;
- recomendações inteligentes;
- otimização de custos;
- aplicativo mobile;
- sincronização online.

Essas funcionalidades pertencem às versões futuras.

---

# Filosofia

O sistema deve separar claramente:

- dados;
- cálculos;
- apresentação.

Nenhum cálculo será executado diretamente na interface do usuário.

Todo cálculo deverá ser centralizado no Core Engine.
