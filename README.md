# 🌱 Calc-Grower Pro

## Sobre

Calc-Grower Pro é uma plataforma de formulação nutricional para cultivo em fibra de coco.

O objetivo é criar uma ferramenta profissional para:

- Formulação de fertilizantes
- Cálculo automático de ppm
- Estimativa de EC
- Manejo de irrigação
- Diagnóstico de runoff
- Banco de genéticas
- Histórico de cultivos
- Controle de custos
